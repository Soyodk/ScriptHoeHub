# ScriptHoeHub - Universal Loader

### How to Use

1. **Join a Supported Game**  
   Open Roblox and join any game supported by ScriptHoeHub.

2. **Execute the Loader**  
   In your Roblox executor, paste and run the following code:

   ```lua
   loadstring(game:HttpGet("https://raw.githubusercontent.com/Soyodk/ScriptHoeHub/refs/heads/main/main.lua"))()
   ```

3. **Done**  
   The loader will automatically detect the game you are playing and load the correct script for it.

---

### Important Information

- This loader works for **all supported games** in the ScriptHoeHub.
- You must be **inside a compatible game** for the script to load properly.
- If the game is supported, you will see a "Game Detected" notification and the script will start.
- If the game is not supported, you will receive a warning message.

### Recommended Executors
- Wave
- Fluxus
- Delta
- Any executor that supports `HttpGet` and has good stability.

### Troubleshooting

- No response? Make sure you are in a supported game and re-execute the loader.
- Errors appearing? Try rejoining the game and executing again.
- "WindUI not found"? Some games may require the UI library to be loaded first.
